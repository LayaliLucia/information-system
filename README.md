# 📊 Information System Project

This repository contains my **Information System project**, developed as part of my coursework.  
The project goal was to **create a Business Requirements Document (BRD)** and present a **PowerPoint** to our professor.

## ✨ Contents
- 📄 **Information System Final Draft (BRD)** – detailed documentation of requirements
- 📊 **Information System Presentation (PPTX)** – slides summarizing the project
- 📘 **README.md** – project overview

## 🎯 Purpose
The aim of this project was to:
- Practice writing a structured **BRD document**
- Translate requirements into a clear, professional presentation
- Demonstrate understanding of information systems concepts
- Present findings effectively to our professor

## 📌 Notes
This project represents both **documentation and presentation skills**, combining technical analysis with communication.  
It was created with ❤️ as part of my academic journey.
